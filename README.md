# sigfried's dotfiles. 

## Contain command prompt utilities, command prompt customization and some useful scripts.
![](https://i.imgur.com/0431KrD.png)

## How to install:
```bash
~ > git clone https://github.com/sigfriedCub1990/dotfiles/
~ > cd dotfiles
~/dotfiles > ./install
```
